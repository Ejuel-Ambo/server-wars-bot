================================================================================
SERVER WARS: DOMINION PROTOCOL - COMPLETE COMMAND STRUCTURE
================================================================================

This document lists ALL commands organized by category, with descriptions,
synergies, and strategic purpose. Every command must contribute to game loop.

================================================================================
COMMAND DESIGN PRINCIPLES
================================================================================

1. PURPOSE: Every command has clear strategic value
2. SYNERGY: Commands interact and create strategic depth
3. NO DEAD COMMANDS: No novelty or pointless commands
4. CLARITY: Commands are intuitive and well-documented
5. BALANCE: Commands are balanced for fair gameplay

================================================================================
CATEGORY 1: REGISTRATION & ONBOARDING
================================================================================

/nation_register <nation_name>
Permission: Server Admin
Purpose: Register Discord server as a nation
Effect: Creates nation in database, sets leader, grants starting resources
Synergies: Enables all other nation commands
Phase: 1 (Foundation)

/nation_info [nation_name]
Permission: Guest
Purpose: View detailed information about a nation
Shows: Name, leader, resources, territory, military, alliances, wars
Synergies: Intelligence gathering, target selection
Phase: 1

/nation_rename <new_name>
Permission: Leader
Purpose: Change nation name
Cooldown: 30 days
Cost: 1000 resources
Phase: 2

/citizen_register
Permission: Guest
Purpose: Register as a citizen of the server's nation
Effect: Enables contribution and participation
Synergies: Unlocks citizen commands, increases nation population
Phase: 1

/citizen_info [user]

/tutorial
